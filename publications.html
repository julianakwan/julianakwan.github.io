Galaxy Clustering in the Mira-Titan Universe 1: Emulators for the redshift space galaxy correlation function and galaxy-galaxy lensing

    In this paper, I use the Mira-Titan simulation suite of 111 cosmologies to build a surrogate model using Gaussian Process regression. The Mira-Titan suite is a set of high resolution, large volume simulations that are designed to span an 8 dimensional cosmological parameter space with the goal of using these models as a training sample to predict clustering statistics as a function of cosmology. In this study, I populate each simulation with a set of mock galaxies using a technique called Halo Occupation Distribution (HOD) modelling, in which the galaxy population is determined by the host halo mass. Then to make predictions for any arrangement of parameters, whether HOD or cosmological, I use Gaussian Process regression to quantify the non-linear relationships between these variables. This gives us direct access to observable predictions from these N-body simulations (to within a few percent accuracy) without having to run the simulations themselves.     Cosmic Emulation: Fast Predictions for the Galaxy Power Spectrum

    How can we make fast and accurate predictions of the non-linear structure formation captured by N-body simulations? In this paper, I populate a high resolution N-body simulation with synthetic galaxies to produce a number of mock galaxy catalogues. I was able to use Gaussian Process regression to predict the galaxy two-point correlation functions with only 1\% error, while varying the galaxy properties over a five dimensional parameter space.
